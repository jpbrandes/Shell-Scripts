# Shell-Scripts
Shell Scripts documentation. Used to automate tasks on Unix systems.

---

## Hardware1.sh

Contains a hardware and software integrity check script, which can be easily used in any terminal environment due to its simple and easy-to-navigate CLI interface.

---

## Hardware2.sh

It's the same script, but with a subtle change in the APT call command, the first one uses apt, while this second one uses apt-get, ideal for compatibility with any Linux version and perfect for scripts.

---

## EssentialPrograms.sh

A simple script that installs useful programs, the programs contained in the script are:
- Gimp
- Libreoffice
- VLC
- Google Chrome

---

## ScriptInstall.sh

Used to automate GitHub configuration, so that it doesn't ask for a password. More advanced environment preparer with compilers included, LaTeX installation and a list of commands taken from the official GitHub documentation for automating tasks for GIT.
- Link with the commands used in the script: https://github.com/cli/cli/blob/trunk/docs/install_linux.md#debian
